# Deploying to production
```bash
cd 1egoman/lunchbox-webapp
biome use lunchbox-webapp-production # sets HOSTNAME and TOKEN env variables
npm run build
npm run deploy
# open https://1egoman.github.io/lunchbox-webapp in a web browser!
```
